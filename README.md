简单写一下，一个具有可视化页面的频道同步和复制工具，使用Pyrofork，支持复制历史信息
<img width="1466" height="891" alt="image" src="https://github.com/user-attachments/assets/9d38b7b8-2bca-47da-ab30-bbf0ca15f241" />
全部代码由AI写的，运行：下载本仓库，在仓库根目录创建py虚拟环境，虚拟环境内安装requirements.txt 内的依赖，填写申请的TGapi和用于复制历史消息的用户账号，终端内运行python main.py 即可
介于tg很容易封禁，所以账号不要使用你的大号
